# 📝 Todo Manager & Signup Validation Website

A simple **JavaScript DOM project** that demonstrates:

* Creating and managing a **Todo List**
* Handling **DOM events**
* Performing **Form Validation using JavaScript**
* Using **HTML, CSS, and Vanilla JavaScript**

This project is great for beginners learning **JavaScript DOM manipulation and form validation.**

---

# 🚀 Features

## 1️⃣ Todo List

Users can manage their daily tasks.

**Functions**

* Add a new todo
* Mark a task as completed
* Remove a task

**How it works**

* User enters a task
* Clicks **Add Todo**
* Task appears in the list
* User can:

  * Click **Done** → Task becomes strikethrough
  * Click **Remove** → Task is deleted

---

## 2️⃣ Signup Form Validation

The signup form includes **client-side validation using JavaScript**.

### Validations Implemented

| Field            | Validation                                     |
| ---------------- | ---------------------------------------------- |
| Username         | Cannot be empty and must not contain `@`       |
| Password         | Minimum 6 characters                           |
| Password         | Must contain at least one **uppercase letter** |
| Password         | Must contain a **special character (@ # $)**   |
| Confirm Password | Must match password                            |
| Phone            | Must be **exactly 10 digits**                  |
| Email            | Must be a valid **Gmail address**              |
| About Yourself   | Minimum **20 characters**                      |

If any validation fails, the user receives an **alert message**.
